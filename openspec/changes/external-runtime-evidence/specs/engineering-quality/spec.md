# External Runtime Evidence Delta

## ADDED Requirements

### Requirement: SQLite is proven through the packaged WebView boundary

The project SHALL provide a deterministic smoke that performs CRUD through the Tauri SQL plugin from the real application WebView and reports a machine-readable result.

### Requirement: Hosted delivery gates are observed and enforced

The verified commit SHALL be present on the configured remote, hosted workflows SHALL pass, and `main` protection SHALL require the resolved quality checks where repository permissions permit.

### Requirement: Native audio evidence is explicit

The native application SHALL safely probe device enumeration, capture/drone lifecycle, typed status, and teardown. Permission prompts and acoustic correctness SHALL be reported as physical evidence when automation cannot prove them.

### Requirement: macOS packaging evidence is decomposed

Application bundle creation, code signing, entitlements, hardened runtime, and notarization SHALL be reported as separate evidence rather than collapsed into a single build claim.
