## Notes

**How to Use This File:**

1. **Starting New Feature:**
   - Add to "Active Implementation" with MVP scope
   - Identify Phase 2 items early

2. **Completing MVP:**
   - Move to "MVP Complete (Phase 2 Backlog)"
   - List all deferred Phase 2 items
   - This prevents feature amnesia!

3. **Phase 2 Implementation:**
   - Check off items as completed
   - Update status when fully complete

4. **Archiving:**
   - Move fully completed features to "Archived"
   - Include completion date

**Why This Matters:**
- Prevents forgetting about Phase 2 work after MVP
- Tracks feature dependencies
- Shows project evolution over time
- Helps with planning and prioritization
